# MoodMusic
L'applicazione Consiglia una playlist basandosi sul mood inserito dall'utente e sul meteo della città inserita
##Funzionalità
L'applicazione permette il funzionamento in 2 modalità:
1. con l'inserimento di un API key di OpenWeather verifica temperatura e condizione atmosferica
2. senza inserire un API Key, l'applicazione entrerà in modalità "demo" e restituirà temperatura e condizione atmosferica ipotetica
L'applicazione successivamente calcolerà una "vibe" basandosi sui dati ricevuti dal calcolo precedente e la abbinerà al "mood" inserito dall'utente per restituire la playlist più adatta
